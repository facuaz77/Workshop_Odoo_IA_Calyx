Requerimientos Técnicos:

Debe depender explícitamente del módulo base y sale ('depends': ['base', 'sale']) para asegurar que se instale correctamente sobre Ventas.

Importante: NO generar archivos de documentación como README.md por el momento.

Modelo (sale.order):

Heredar sale.order.

Agregar al campo state (Selection) una nueva opción: clave 'acopio', etiqueta "Acopio".

Asegurar que este nuevo estado se refleje en el statusbar de la vista formulario.

Wizard (TransientModel):

Crear un wizard llamado construnort.acopio.wizard.

Incluir un campo Boolean (ej. check_process) que el usuario debe marcar.

Incluir un botón/método "Confirmar" que valide el booleano y cambie el state de la orden activa a 'acopio'.

Vistas (XML):

En sale.order: Agregar un botón en el <header> con el texto "Acopio".

Este botón debe abrir el wizard creado arriba.

Visibilidad: El botón solo debe ser visible si el usuario pertenece al grupo construct_acopio.

Seguridad:

Crear el grupo de seguridad construct_acopio.

Generar el archivo ir.model.access.csv para dar permisos al wizard.
